State first design

Reduces unknown unknowns that impact fluency, focus, motivation, collaboration, morale, speed and quality

Getting Started

  • Not easy... but you're going to do it anyway
  • Visual examples better then theory
  • Everything is State+Action=New State
  • Ignore impossible states e.g. Can't be loading and error at the same time
  • Divide and conquer! Start with a small simple state and build from there
  • Collaborate and share
